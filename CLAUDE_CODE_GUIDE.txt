============================
  Claude Code 快速啟動指南
============================

1. 打開終端機

2. 進入專案資料夾：
   cd baseball-bot

3. 啟動 Claude Code：
   claude

4. 直接用中文輸入需求即可，例如：
   「幫我加一個追蹤的選手」
   「投手推播加上勝敗紀錄」

5. 離開：
   /exit

============================
  首次安裝（只需做一次）
============================

npm install -g @anthropic-ai/claude-code
